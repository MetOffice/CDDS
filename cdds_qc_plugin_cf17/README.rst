CDDS QC CF1.7 Plugin for Compliance Checker
===========================================

.. include:: common.txt

The CF1.7 plugin is an extension of the original CF1.6 checker, providing
some additional features and configurability.

An overview of CDDS QC CF1.7 Plugin
-----------------------------------

The plugin overrides some methods to include checks related to CF1.7
compliance, fixes some bugs, and introduces a capability to configure the
plugin during the runtime.
