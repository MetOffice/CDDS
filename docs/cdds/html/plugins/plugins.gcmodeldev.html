class cdds.common.plugins.gcmodeldev.gcmodeldev_models.GCModelDevStore[source]¶

Bases: cdds.common.plugins.base.base_models.BaseModelStore

Singleton class to store for each model the corresponding parameters. The parameters are defined in json files. The default parameters are stored in the data/model directory.

The class is a singleton to avoid excessive loading of the parameters from the json files.
