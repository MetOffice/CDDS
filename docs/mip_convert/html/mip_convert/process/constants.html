constants¶

The constants module contains the constants available for use in the model to MIP mapping expressions.

mip_convert.process.constants.constants()[source]¶

Return the names and values of the constants available for use in the model to MIP mapping expressions.

Since the values of the constants are used to replace the names of the constants in the model to MIP mapping expressions, the values are stored as strings.

Returns

The names and values of the constants available for use in the model to MIP mapping expressions.

Return type

dictionary
