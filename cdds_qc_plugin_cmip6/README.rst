An overview of CMIP6 Compliance Checker Plugin
----------------------------------------------

This package is the CMIP6 standards compliance checker plugin for
`ioos/compliance-checker <https://github.com/ioos/compliance-checker>`_.

The checker provides a suite of tests related to CMIP6 compliance, including
global attributes and variable attributes validation, and checking their
consistency against the CMIP6 |Controlled Vocabulary|.
