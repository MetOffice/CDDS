load¶

The load package enables the input variables for a MIP requested variable to be loaded from the model output files.

mip_convert.load.load(filenames, variable_metadata)[source]¶

Return the data and metadata related to a MIP requested variable.

Parameters

  • filenames (list of strings) – The filenames (including the full path) of the files required to produce the output netCDF files for the MIP requested variable.

  • variable_metadata (new_variable.VariableMetadata) – The information required to load the appropriate data from the model output files to create the input variables for the MIP requested variable.

Returns

The data and metadata related to a MIP requested variable.

Return type

new_variable.Variable
